# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-10-07

### Added
- `@automation-blueprints/dsl` package — Blueprint validation SDK with JSON Schema Draft-07
- `@automation-blueprints/adapters` package — Platform export adapters for Zapier, Make, n8n, Power Automate
- Blueprint examples: basic (HubSpot-Slack), intermediate (Stripe invoice notifications), advanced (lead-to-contract, GitHub-Notion tracker)
- Platform export examples showing the same blueprint in Zapier, Make, and n8n formats
- SDK usage examples with TypeScript
- DSL documentation: breaking change policy, migration guide
- Blueprint Contributor Agreement
- Security policy
- Apache 2.0 License
